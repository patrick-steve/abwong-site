# Design Context

Strategic and visual context for this project lives at the repo root:

- `./PRODUCT.md` — register, audiences, brand personality, anti-references, design principles, accessibility commitments.
- `./DESIGN.md` — visual system in Stitch DESIGN.md format: tokens (frontmatter), colors, typography, elevation, components, do's and don'ts.
- `./.impeccable/design.json` — sidecar with tonal ramps, motion vocabulary, and full HTML/CSS snippets per component.

Read both PRODUCT.md and DESIGN.md before designing or building any UI. The DESIGN.md "Don'ts" section is the load-bearing list of anti-patterns; treat them as hard rules, not suggestions.
